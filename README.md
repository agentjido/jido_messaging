# Jido Messaging

Messaging and notification system for the Jido ecosystem.

## Installation

The package can be installed by adding `jido_messaging` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:jido_messaging, "~> 0.1.0"}
  ]
end
```
